JavaScript Interno

javaScript Interno é quando trabalhamos com o código JS diretamente no arquivo HTML dentro da tag <script>. Geralmente, essa tag <script> costuma ficar próxima a tag de fechamento do <body>.

JavaScript Externo

No JS Externo, trabalhamos com o código JS em um arquivo separado, Esta é a rática recomendada, pois isso garante que a estrutura do HTML fique mais organizada.

Para usarmos um arquivo externo, basta adicionar o atributo src configurado na tag <script>, geralmente próximo a tag de fechamento <body>.
